# Чеклист перед запуском рекламы

## Обязательные шаги

- [ ] Заменить `XXXXXXXXXXXXXXXX` на реальный Meta Pixel ID (в admin-director.html → Контакты и настройки → «Meta Pixel ID»)
- [ ] Задеплоить Cloudflare Worker (см. `workers/README.md`)
- [ ] Вставить URL Worker в `website/js/main.js` — константа `WORKER_URL` в начале файла (заменить `ТВОЙ_АККАУНТ` на реальный поддомен)
- [ ] Проверить форму: заявка приходит в Telegram (отправить тестовую заявку через сайт)
- [ ] Установить PIN для `admin-director.html` (первый вход — страница предложит его создать)
- [ ] Добавить реальные цены через admin-director.html → Цены
- [ ] Добавить реальные отзывы через admin-director.html → Отзывы

## SEO и локальные каталоги

- [ ] Зарегистрировать Google Business Profile (Петропавловск, категория «Школа искусств» / «Центр детского развития»)
- [ ] Зарегистрировать на 2GIS (бесплатная карточка организации)
- [ ] Добавить сайт в Яндекс.Справочник
- [ ] Отправить `sitemap.xml` в Google Search Console (URL: `https://artshpace.github.io/bot/website/sitemap.xml`)

## Контент

- [ ] Загрузить реальные фото преподавателей (заменить заглушки в media/)
- [ ] Обновить карточки преподавателей в admin-director.html → Преподаватели
- [ ] Проверить актуальность расписания слотов (admin-director.html → Расписание пробных)
- [ ] Уточнить и при необходимости обновить Telegram-хэндл (`t.me/shpigotskiy_art`)

## Перед запуском Instagram-рекламы

- [ ] Убедиться, что форма записи работает (заявка → localStorage → Telegram)
- [ ] Протестировать WhatsApp-переход после отправки формы
- [ ] Проверить страницы направлений на мобиле (особенно guitar, acting — основной трафик)
- [ ] Убедиться, что admin-director.html открывается только по PIN

## Технические

- [ ] Закоммитить все изменения в репозиторий (ветка `claude/shpigotskiy-art-space-site-2uwim5`)
- [ ] Убедиться, что GitHub Pages деплоит актуальную версию
- [ ] Проверить 404-ошибки в консоли браузера (особенно пути к media/)
