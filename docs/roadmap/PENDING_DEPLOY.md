## 1. Один передеплой Worker `sas-lead-forwarder` закрывает СРАЗУ три вещи
Обнови код Worker'а свежим `workers/lead-forwarder.js` (Edit code → выделить всё →
вставить → Deploy). Этого достаточно для:

- **Воронка (Задача 11):** лиды с сайта начнут писаться в таблицу `leads` →
  появятся в «Админ · Лиды» и «Воронке». (Миграция 0014 уже применена.)
- **Напоминания (Задача 3):** заработает кнопка **«📨 Напомнить»** в журнале —
  рассылает сообщение в Telegram ученикам группы и их родителям (у кого привязан бот).
  Эндпоинт `/notify`. Доп. переменные не нужны.
- **Google-календарь (Задача 5):** заработает, **если** добавить переменные из п.2.

Проверка после деплоя:
- Заполни заявку на сайте → смотри «Админ · Лиды» (лид появился) и «Воронку».
- В журнале выбери группу с привязанным учеником → «📨 Напомнить» → придёт в Telegram.

## 2. Google-календарь директора (Задача 5) — настройка Google Cloud
Нужно один раз настроить сервисный аккаунт и расшарить календарь.

1. **Google Cloud Console** → создай проект → включи **Google Calendar API**.
2. **Создай Service Account** → в нём **Keys → Add key → JSON** → скачается файл с
   приватным ключом. Это СЕКРЕТ.
3. В **Google Календаре** (под аккаунтом, где должны появляться события) → Настройки
   нужного календаря → «Доступ для отдельных пользователей» → добавь **email
   сервисного аккаунта** (из JSON, поле `client_email`) с правом **«Внесение
   изменений в мероприятия»**.
4. В Cloudflare → Worker `sas-lead-forwarder` → Settings → Variables, добавь:
   - `GOOGLE_SA_EMAIL` (Plain) — `client_email` из JSON
   - `GOOGLE_SA_PRIVATE_KEY` (**Secret**) — `private_key` из JSON целиком
     (вместе с `-----BEGIN PRIVATE KEY-----` … `-----END PRIVATE KEY-----`).
     Можно вставлять с `\n` — код сам их развернёт.
   - `GOOGLE_CALENDAR_ID` (Plain) — id календаря (обычно это его email, напр.
     `artshpace@gmail.com` или строка вида `...@group.calendar.google.com`).
5. Deploy. Теперь каждый новый лид со слотом → событие в календаре на ближайший
   нужный день/время (Asia/Almaty). Слоты без распознанного времени → событие на
   весь день с пометкой «согласовать время».

⚠️ JSON-ключ и `GOOGLE_SA_PRIVATE_KEY` — секреты: только в Cloudflare как Secret,
никогда в чат и не в репозиторий.

Проверка: заполни заявку с конкретным слотом (напр. «Среда, 17:00–18:00») → в
календаре директора в ближайшую среду появится событие «Пробное — Гитара, Имя».
